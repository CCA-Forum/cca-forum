Ccaffeine Build Example for cccaffeine-09-06-2002.tgz


This is an example build for Ccaffeine from my laptop.  The paths that you give to "configure" are going to be different, but it will give you an idea as to what it takes.  It will configure and build using the normal previous procedure.  It must have two ancillary repositories to build: cca-spec-classic, and cca-spec-babel.  The *real* cca spec is the babel one.  These must be built before Ccaffeine is built.  Here is a list of the things required:

  • Babel
  • BOOST smart pointers
  • BLAS and LAPACK
  • Ruby (ships with Redhat now, but let me know if this is a hassle, I can fix this)
  • gcc 2.95.3 (3.1 may work, untested)
