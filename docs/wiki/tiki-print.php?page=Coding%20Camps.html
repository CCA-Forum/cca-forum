Coding Camps

Usability Coding Camps

Chicago May 3 - May 5, 2005


Working on implementation of

  • Eclipse-based build tools for multi-language software projects.

  • Bourne shell/make-based tools for better build and source code management using Babel and/or CCA.

  • Generation of complete dependency information in Babel make or other build tools. The idea is to generate makefile fragments that contain dependency information for targets generated by Babel. A new class, gov.llnl.babel.backend.Dependencies was introduced. This class implements the FileListener and BuildGenerator interfaces, and uses metadata associated with SIDL symbols to produce the target and prerequisites information needed for dependency generation.


We've put together a Usability Work List that we expect to grow. Much of it initially is oriented to the build issues.
