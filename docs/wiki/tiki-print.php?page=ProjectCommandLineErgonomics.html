See also Ben Rob Toolkit Tools Discussion for further commandline discussion.

Rob is thinking Torsten-like thoughts about commandline tools which would let you in shell do things like
defineProject mytest
cd mytest_dir
defineComponent Foo cxx
cd Foo_dir
addProvidesPort ccafe.StringConsumerPort
registerUsesPort ccafe.TimePort
genBuildScripts
make
make test
The gorey details of which could easily run to 100s of pages.

Of course there are lots of other ways the same set of transformations implied above could be wrapped up in scripting languages or in a single tool with a lot of options. We'll need our own splicer to make something like the above work, but we needed it anyway to make the mittens work.
