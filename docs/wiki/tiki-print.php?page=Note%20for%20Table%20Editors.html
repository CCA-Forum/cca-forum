Note for Table Editors

work around for tikiwiki table bug

Tables in this wiki *must* be entered as a single line of text.
Inserting extra whitespace will cause the text to appear sanely in the edit window while
still being a single line due to the word-wrap.
Be sure to check the saved table for accidental insertion of carriage returns that break the table.
