CCA Tutorial

2009-08-21 at the Tenth Workshop on the DOE Advanced Computational Software (ACTS) Collection

On 2009-08-21, we presented an extended tutorial at the ACTS Workshop, including a 90 minutes of lecture material and a 3 hour hands-on portion.

PDF files are 2-up handout versions, rendered in color. See the CCA tutorials home page to obtain the latest hands-on materials associated with this tutorial.

Some important notes about the files in this archive:

  • Use cca-tools-contractor-acts09 to build the CCA tools.
    • This version of contractor is configured to download the tools from http://www.cca-forum.org/download/tutorial/cca-tools-acts09/
    • This copy of cca-tools-acts09 is for archival purposes only, you do not need to download it in order to build the CCA tools
  • The Bassi version of the tutorial source code is for archival purposes only, you should not expect it to work on other platforms.
