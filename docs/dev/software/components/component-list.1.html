CCA Toolkit Components

Prepackaged components that work with CCA Tools

 

CCA Components come as source code tar balls for now.   They must be built on your machine using your already built CCA Tools.  Click here for more information regarding CCA Tools building and installing.  It is important to read the documentation for each download to make sure that you have satisfied its dependencies.  Beyond external dependencies, the component must acquire information about your CCA Tools install: usually putting the ccafe-config executable in your path will be enough.

Description Title
LInear Solver Interface CCA LISI
GA Component encapsulates parallel distributed array functionality in component form Global Arrays
Intercomm is a framework that facilitates interoperation between two parallel codes. The Intercomm component does the same for CCA components. Intercomm Parallel Coupler
A Parallel Rectilinear Grid Component Parallel Cartesian Grid
SPARSKIT suite of components from SPARSKIT2. SPARSKIT Suite
Toolkit for Advanced Optimization (TAO) in CCA component form. TAO Numerical Optimization Components
Tuning and Analysis Utilities (TAU) as a CCA component. TAU Components
TOPS Interfaces and Implementation Based on PETSc TOPS/PETSc
