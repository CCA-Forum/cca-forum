SPARSKIT Suite

by bek — last modified Jan 14, 2009 06:51 PM
— filed under: cca component

SPARSKIT suite of components from SPARSKIT2.

Name: SKITtk

Author: Masha Sosonkina

Description:

The SPARSKIT Toolkit is a suite of components that uses the CCA Tools software to accomodate component connection. The components are derived from SPARSKIT2, a basic tool-kit for sparse matrix computations written in FORTRAn77 and developed by Yousef Saad at the University of Minnesota.

The original SPARSKIT2 is available from:

        ftp://ftp.cs.umn.edu/dept/sparse/SPARSKIT2.tar.gz.

The SPARSKIT CCA Toolkit  takes the original implementation of SPARSKIT and implements components around the major parts, including accelerators, preconditioners, matrix generation, matrix-vector multiplication, and matrix I/O.
