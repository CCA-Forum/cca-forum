Platforms for CCA Toolchain testing
At the moment, this is a list of platforms of interest not a list of platforms we are actually testing on.

Cross-Configure/Cross-Compile Platforms

  • Cray XT/Compute Node Linux
    • Sites: ORNL, NERSC
    • Owner: Jim Kohl

  • IBM Blue Gene/P
    • Sites: ANL. ORNL (small)
    • Owner: Boyana Norris

  • IBM Blue Gene/L
    • Sites: ANL, LLNL (no unclassified access at present)
    • Owner: Gary Kumfert
    • Issues: small memory challenging for component apps

Direct-Build Platforms

  • COTS Linux distros (RHEL, Fedora, Ubuntu, etc.)
    • Sites: widely used

  • Mac OS X
    • Sites: less widely used

  • Solaris
    • Sites: UC-Boulder (outside access hard)

  • AIX
    • Sites: NERSC (bassi)
    • Owner: Ben Allan ???

  • MPP2 (To be replaced by "MPP3" as soon as it comes online.)
    • Sites: PNNL
    • Owner: Manoj Krishnan ???
    • Issues: Only 2 of >20 combinations of modules will work with Babel

  • MPP3
    • Sites: PNNL
    • Owner: Manoj Krishnan ???
    • Issues: Announced, but not in production

Waiting for Expression of User Need

We do not plan to pursue support for these platforms unless we receive specific user requests, or we see that the platform reaches a reasonable level of use and deployment within our user community.

  • SiCortex
    • Sites: ANL
    • Owner; Boyana Norris
    • Issues: Requires Chasm to support PathScale compilers. Sysadmins unwilling to add software beyond what SciCortex supplies. New platform, need evidence of community interest.

  • Cray XT/Catamount
    • Sites: SNL
    • Owner: Ben Allan
    • Issues: One-off OS. Support only if needed.
