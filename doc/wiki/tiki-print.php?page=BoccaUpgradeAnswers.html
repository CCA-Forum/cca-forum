Upgrading from one release to another

The bocca developers are deeply aware that difficult migration between versions of a support tool is a prime reason for rejection of that tool. We do everything we can in formal releases to maintain compatibility with or a simple upgrade path from the prior formal release. As of Oct 17, 2007, there are no formal releases. We expect one very soon, however, 0.4.2.

More details on converting between specific releases will be in this section as releases are made. Those who live on the bleeding edge (using bocca source from its subversion repository) may need to make a more complex effort upgrading from some snapshot to a formal release. See BoccaSnapshotUpgrade for simple strategies to handle this hard case.
