Current Progress on Developing a Subsurface Simulation Framework Using CCA
Bruce Palmer, Yilin Fang, Glenn Hammond, Vidhya Gurumoorthi, Jim Fort (PNNL)

We will review recent progress on creating a component framework for subsurface simulations based on the smoothed particle hydrodynamics (SPH) algorithm. Current efforts to develop a similar set of components for continuum simulations built around the STOMP subsurface simulation code will also be discussed. We will also describe the main aspects of our strategy for creating a relatively modular, plug-and-play set of components that contain minimal dependencies on other components. This includes the creation of a universal “dictionary” of data objects, the use of a data manager, and the use of individual input to separate components.

Approaches and Cost Estimation for Supporting Microsoft
Gary Kumfert (LLNL)

Gary Kumfert and Tom Epperly spent 3 days at Microsoft's Platform Adoption Center evaluating strategies for supporting Microsoft platforms in Babel. We learned some new tricks there and have assessed CCA's options for adding Microsoft platforms to our list of supported platforms. This talk will enumerate available technical approaches and compare the efforts involved with each approach.
