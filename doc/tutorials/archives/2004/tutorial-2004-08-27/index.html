CCA Tutorial

2004-08-27 at Fifth Workshop on the DOE Advanced Computational Software Collection

On 2004-08-27, we presented our "full day" tutorial at the ACTS Workshop, including a hands-on portion.

PDF files are 2-up handout versions, rendered in color. See the CCA software page to obtain the latest version of the example software referenced in this tutorial as well as other CCA tools and components. Participants can retrieve their personal source code files as gzipped tar files, where the files are named by the student account number used.
